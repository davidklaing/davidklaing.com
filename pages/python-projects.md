## Set up virtual environment

Create a virtual environment.

```bash
python -m venv venv
```

Activate the virtual environment

```bash
source venv/bin/activate
```

Add venv to `gitignore`

## Create a package

```bash
mkdir {package_name}
```

Add `__init__.py` to each module.
